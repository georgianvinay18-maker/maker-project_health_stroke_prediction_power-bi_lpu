# maker-project_health_stroke_prediction_power-bi_lpu
Global Health Statistics Dashboard using Power BI 📊
I’m excited to share my academic project “Global Health Statistics”, developed as part of my INT387 – Data Analytics with Power BI course.
This project analyzes global health data from 2000 to 2024, covering 20 countries and 20 diseases, to uncover meaningful insights related to disease trends, demographic impact, and healthcare performance.
🛠 Tools & Techniques Used
• Microsoft Power BI
• Power Query for data cleaning & EDA
• DAX for KPI calculations
• Interactive dashboards with slicers
📈 Key Insights
• Average Mortality Rate: 5.04%
• Average Healthcare Access: 74.90%
• Average Recovery Rate: 74.58%
• Disease impact varies significantly by country, year, age group, and gender
